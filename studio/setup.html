Kit Setup Instructions

Follow one of the two setup flows below depending on whether you're configuring a new system or reconnecting an existing Inventa kit.

🆕 New Kit Setup

  1. Power on the Inventa Hub device
  2. Wait for WiFi network: Inventa-Setup
  3. Connect using phone or laptop
  4. Open browser → 192.168.4.1
  5. Enter your WiFi SSID + Password
  6. Click Save → device will reboot
  7. Open Studio → system will auto-connect
⚡ Tip: If setup fails, restart the hub and reconnect to the setup network.

⚙️ Existing Kit Setup

  1. Power on your Inventa Hub
  2. Ensure it is already connected to WiFi
  3. Open Studio dashboard
  4. Wait for device scan
  5. Select your hub from available devices
  6. Click Connect
  7. Begin interaction / control mode
🔗 Tip: If device is not found, ensure same network or restart hub.
