Inventa Hub Setup

System Status Indicators

Hub Connection Setup

Follow this guide to connect your Inventa Hub to a network and enter Studio mode. The hub uses a 3-state system: White, Red, and Green.

⚪ White — Setup Mode

The hub is ready for configuration. This appears when:

  • First-time setup
  • Previous network not found
  • Manual reset

A WiFi network called “Inventa Setup” will appear.

🔴 Red — Connection Failed

The hub could not connect to the network.

  • Wrong WiFi name or password
  • Hotspot compatibility off
  • Max device limit reached
  • Network turned off

Hub will return to setup mode automatically (White).

🟢 Green — Connected

The hub successfully connected to WiFi.

  • Data link active
  • Studio connection enabled
  • Ready for robotics control

You can now return to Studio.

Setup Flow

1

Turn on the Inventa Hub device.

2

If white mode appears → connect to Inventa Setup network.

3

Open browser and go to 192.168.4.1.

4

Enter WiFi name + password for your home/hotspot network.

5

Hub processes data and attempts connection.

6

If correct → Green light; If wrong → Red → back to White

7

Reconnect to your network and open Studio.

Troubleshooting

If the hub stays red:

  • Check WiFi password accuracy
  • Enable hotspot compatibility mode
  • Ensure device limit is not reached
  • Restart router or hotspot

If hub disconnects suddenly:

  • Network went offline
  • Hub requires restart
  • Re-enter setup mode manually

System will always fallback to White (safe mode).
