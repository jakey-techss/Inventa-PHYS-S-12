Universal Modular Sensor Pod

For this assignment I designed a modular sensor pod that will eventually become part of my larger robotics platform. Instead of creating a simple practice model, I wanted to design something that could realistically be integrated into my final project. The sensor pod is designed as a removable subsystem that can house different sensors while attaching to the robot through a universal mounting bracket

The Assignment

Select at least two household objects or laboratory components, measure them using calipers, recreate them in Fusion 360, and build an assembly consisting of multiple components connected using appropriate joints

Objective

Demonstrate Fusion Parametric design by producing measured tools or components that could be used in the final project

My Approach

Rather than modeling unrelated household objects, I chose to design a modular sensor pod that can eventually become a subsystem of my final robotics project.

The Sensor Pod

The Universal Modular Sensor Pod is a removable subsystem that houses sensing electronics while protecting them from damage. The enclosure contains an internal PCB, camera module and a removable rear panel. The overall design focuses on modularity and future expansion into a complete robotics platform

How I Built It?

Rather than modeling the complete assembly at once I followed a modular workflow by creating each component as its own Fusion component before combining everything into a one assembly. This allowed every part to remain editable while making it much easier to test different design ideas.

01

Research

Collected reference images of standard sensor housings and replaceable robotic vision systems

02

Measurements

Measured physical components using digital calipers to obtain accurate dimensions before beginning CAD.

03

Sketches

Created constrained sketches using parameters and construction geometry to define the overall shape.

04

3D Modeling

Converted sketches into solid bodies using extrudes, fillets, chamfers and shell operations.

05

Assembly

Created individual components before combining them using joints.

06

Rendering

Applied appearances and generated final renders showing the completed assembly.

Engineering Challenges

Parametric Design

Maintaining consistent dimensions between sketches while making the design flexible enough for future modifications.

External Cover

Ensuring every internal component could fit inside the sensor housing without any collision between the different components.

What I Learned

  • Creating fully constrained sketches dramatically improves model reliability
  • Breaking a project into components makes large assemblies much easier to manage
  • Using parameters allows dimensions to update automatically throughout the model

Future Improvements

This CAD model serves as the foundation for my larger robotics project. Future versions will incorporate functional electronics, a custom PCB, modular communication ports and a standardized pogo pin conneection mode that allows the sensor pod to integrate with the Inventa Kit Hub.

Reflection

This assignment gave me a much stronger understanding of how engineers approach mechanical design. Instead of thinking of a product as a single object, I learned to view it as a collection of carefully designed components working together as one system. Throughout the project I became significantly more comfortable using Fusion 360.
