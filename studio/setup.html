First-Time Configuration

New Kit Setup

  1. 1Power on the Inventa Hub device
  2. 2Wait for WiFi network: Inventa-Setup
  3. 3Connect using phone or laptop
  4. 4Open browser → 192.168.4.1
  5. 5Enter your WiFi SSID + Password
  6. 6Click Save → device will reboot
  7. 7Open Studio → system will auto-connect
Tip: If setup fails, restart the hub and reconnect to the setup network.
⚙️

Reconnect Existing Hardware

Existing Kit Setup

  1. 1Power on your Inventa Hub
  2. 2Ensure it is already connected to WiFi
  3. 3Open Studio dashboard
  4. 4Wait for device scan
  5. 5Select your hub from available devices
  6. 6Click Connect
  7. 7Begin interaction / control mode
Tip: If device is not found, make sure your computer and hub are on the same network or restart the hub.
